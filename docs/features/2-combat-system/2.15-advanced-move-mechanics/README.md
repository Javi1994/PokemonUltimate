# Sub-Feature 2.15: Advanced Move Mechanics

> Multi-hit, charging moves - Advanced move mechanics.

**Sub-Feature Number**: 2.15  
**Parent Feature**: Feature 2: Combat System  
**See**: [`../../features_master_list.md`](../../features_master_list.md) for feature numbering standards.

## Overview

Advanced Move Mechanics implements complex move behaviors:

-   **Multi-Hit Moves**: Bullet Seed, Fury Swipes, etc.
-   **Charging Moves**: Solar Beam, Skull Bash, etc.
-   **Two-Turn Moves**: Fly, Dig, Dive, etc.
-   **Recharge Moves**: Hyper Beam, Giga Impact, etc.

## Current Status

-   ✅ **Implemented**: Advanced move mechanics implemented
    -   Multi-hit moves (MultiHitEffect)
    -   Counter/Mirror Coat (CounterEffect)
    -   Protect/Detect (ProtectEffect)
    -   Focus Punch (FocusPunchEffect)
    -   Multi-turn moves (MultiTurnEffect)
    -   Pursuit (PursuitEffect)
    -   Semi-invulnerable moves (SemiInvulnerableEffect)

# Sub-Feature 2.11: Recoil & Drain

> Recoil damage, HP drain effects - Recoil and drain move effects.

**Sub-Feature Number**: 2.11  
**Parent Feature**: Feature 2: Combat System  
**See**: [`../../features_master_list.md`](../../features_master_list.md) for feature numbering standards.

## Overview

Recoil & Drain implements move effects that deal recoil damage to the user or drain HP from the target:
- **Recoil Effects**: Moves like Take Down, Double-Edge
- **Drain Effects**: Moves like Giga Drain, Leech Life
- **HP Drain Calculation**: Percentage-based or fixed HP drain

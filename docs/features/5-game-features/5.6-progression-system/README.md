# Sub-Feature 5.6: Progression System

> Roguelike runs, meta-progression - Progression and roguelike system.

**Sub-Feature Number**: 5.6  
**Parent Feature**: Feature 5: Game Features  
**See**: [`../../features_master_list.md`](../../features_master_list.md) for feature numbering standards.

## Overview

Progression System implements roguelike mechanics and meta-progression:
- **Roguelike Runs**: Individual run progression
- **Meta-Progression**: Permanent unlocks and upgrades
- **Run Management**: Starting, continuing, ending runs
- **Progression Rewards**: Unlocks based on progress

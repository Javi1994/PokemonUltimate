# Sub-Feature 5.4: Inventory System

> Item management and usage - Inventory and item system.

**Sub-Feature Number**: 5.4  
**Parent Feature**: Feature 5: Game Features  
**See**: [`../../features_master_list.md`](../../features_master_list.md) for feature numbering standards.

## Overview

Inventory System manages all items in the game:
- **Item Storage**: Inventory management
- **Item Usage**: Using items (potions, berries, etc.)
- **Item Categories**: Organizing items by type
- **Item Limits**: Stack limits and inventory capacity
